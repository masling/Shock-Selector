# EKD SEO Keyword Strategy for Europe & North America
Version: MVP v1  
Primary language: English  
Secondary locales (phase 2): German, French, Italian  
Brand: EKD

## 1. Goal
This document defines the SEO keyword strategy for the EKD website, focused on industrial buyers and engineers in Europe and North America who need to:
- find the right industrial shock absorber / hydraulic shock absorber
- calculate sizing and energy requirements
- compare models by application, load, speed, stroke, and thread size
- understand selection logic before purchasing

The site should not behave like a general brochure site first. It should behave like:
1. a technical selection platform,
2. a product catalog,
3. an application knowledge base.

## 4. SEO Architecture Principles
1. English should be the primary launch language.
2. Build around commercial + technical intent, not broad educational keywords only.
3. Use one page type per search intent:
   - homepage for category + brand intent
   - selector pages for selection intent
   - product family pages for model/category intent
   - application pages for scenario intent
   - guide pages for parameter / calculation intent
4. Build URLs around clean industrial terms:
   - /shock-absorber-selector/
   - /industrial-shock-absorbers/
   - /hydraulic-shock-absorbers/
   - /applications/vertical-motion/
   - /guides/energy-per-cycle/
5. Each important keyword cluster should map to a unique landing page.
6. Product detail pages must include parameter-rich structured content, not just short specs.

## 14. Keywords to Avoid as Main Targets
Do not make these primary homepage targets:
- damper
- shock absorber only
- absorber only
- buffer
- industrial buffer
- machine damper
- vibration damper

Reason:
These terms are broader, more ambiguous, or often map to different product classes.

Use them only as support terms where context is clear.
